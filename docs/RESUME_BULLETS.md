# Resume and portfolio copy

## Project name

PathwiseIQ — a study planner with a product analytics dashboard on top of it.

Stack: React, Tailwind, FastAPI, SQLite, SQL, pandas.

## Resume bullets (pick 2 or 3)

* Built a study planner web app, instrumented the user journey, and made a small analytics dashboard to answer questions like where users drop off and what to fix next.
* Drew a conversion funnel from landing to first completed task, found that step 2 of onboarding was losing roughly 27% of users, and proposed a simpler 2-step flow.
* Ran an A/B test comparing two onboarding lengths, used a two-proportion z-test, and identified a variant that converted about 8.6 points higher (p = 0.013).
* Wrote a basic PRD before coding, scored features with RICE to decide what to build first, and documented what I'd change based on what the data showed.
